# 📊 Análise de Vendas — Olist E-commerce

Análise exploratória de dados reais de e-commerce brasileiro usando Python e pandas.

## 🎯 Objetivo
Extrair insights de negócio sobre crescimento, receita e logística da Olist
a partir de 99.441 pedidos reais entre 2016 e 2018.

## 🔍 Principais Insights
- 🚀 Crescimento de **1.800x** em pedidos em pouco mais de um ano
- 💰 Pico de receita de **~R$ 1.000.000** em novembro de 2017
- 🗺️ **SP concentra 3x mais receita** que o 2º colocado (RJ)
- 🚚 Tempo de entrega varia de **8 dias (SP)** a **18 dias (BA)**

## 🛠️ Tecnologias
- Python 3.13
- pandas
- matplotlib
- Jupyter Notebook

## ▶️ Como rodar
1. Clone o repositório
2. Crie o ambiente virtual: `python -m venv venv`
3. Ative: `venv\Scripts\activate`
4. Instale as dependências: `pip install pandas matplotlib seaborn jupyter notebook`
5. Baixe os dados em: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
6. Execute o notebook: `python -m notebook`

## 📌 Fonte dos dados
[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
